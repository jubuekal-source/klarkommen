# Gedanken-Experiment | Klarkommen

Interaktives Browser-Experiment für **Topic 3 – Gedankenkontrolle** aus dem Programm *Klarkommen – Das Update, das dir fehlt*.

## Was ist das?

Drei Experimente in einem Erlebnis:

1. **Zitronen-Experiment** — Audio-geführte Visualisierung, die zeigt: Dein Gehirn unterscheidet nicht zwischen Vorstellung und Realität.
2. **Luftballon-Experiment** — Ideomotorischer Effekt: Deine Gedanken bewegen deinen Körper, ohne dass du es willst.
3. **Sorgen- & Content-Rechner** — Wie viel deiner wachen Lebenszeit verbringst du im Stressmodus?

## Technologie

Vanilla HTML/CSS/JS — keine Frameworks, keine externen Abhängigkeiten. Web Audio API für Sound-Effekte. Optimiert für Mobile.
