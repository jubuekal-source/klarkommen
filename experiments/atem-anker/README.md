# Atem-Anker

Interaktive Atemübung mit zwei Techniken für den Nervensystem-Reset:

- **Physiological Sigh** – 1-Atemzug-Reset für akute Stresssituationen (Stanford/Huberman Lab)
- **4-7-8 Atmung** – Tiefe Beruhigung für Feierabend und Einschlafen (Dr. Andrew Weil)

Animierter Atemkreis mit Timer, Zyklen-Zähler und wissenschaftlichem Hintergrund.
